GGX / QUEPON NEW MASTER SYSTEM
1. Product Requirements Document
Product Name
QUEPON
Full Title
QUEPON: A Web-Based Computer Shop Management System with Smart Queueing, AI Assistance, Real-Time PC Monitoring, Session Tracking, Wallet Credits, Menu Ordering, and Admin Control for GGX Computer Shop
Product Vision
QUEPON is a smart computer shop management system designed to make GGX more organized, transparent, and efficient. It replaces manual PC tracking, verbal queueing, and scattered communication with one centralized platform where players can check PC availability, join queues, manage sessions, view promos, order from the menu, and receive assistance.
For administrators, QUEPON provides a real-time control center for PC monitoring, queue approval, session assignment, wallet top-ups, promo management, feedback handling, reports, and incident tracing.
Product Goal
To create a user-centered and admin-controlled system that improves the computer shop experience through real-time visibility, structured queueing, automated session handling, and simplified shop operations.
Target Users
Player
A customer who uses GGX Computer Shop services.
Main needs:
See available PCs before asking staff
Join the queue without confusion
Know their queue position
Check their assigned PC
Track remaining session time
Extend session when possible
View promos and shop updates
Order food or services
Send feedback or report issues
Administrator
The staff or shop operator who manages GGX operations.
Main needs:
Monitor all PCs in real time
Approve queue entries
Assign players to PCs
Start, extend, lock, or end sessions
Set PCs as available, occupied, or maintenance
Manage player accounts
Add wallet credits
Update promos and announcements
View feedback and reports
Investigate flagged incidents using logs
Main Problem
GGX currently faces problems caused by manual and informal shop management.
User Problems
Players do not know which PCs are available
Players rely on staff for updates
Queueing can feel unclear or unfair
Session details are not easily visible
Promos, rates, and updates are not centralized
Admin Problems
Staff manually monitor PCs and users
Queue handling becomes harder during peak hours
Repetitive questions consume staff time
Session assignment can be inconsistent
Records are harder to track without a central system
System Problems
No centralized management dashboard
No real-time PC visibility
No structured session flow
No automated queue status
No strong error prevention and feedback system
Proposed Solution
QUEPON introduces a centralized web-based platform with two main systems:
Player App
Designed for customers
Mobile-first and kiosk-friendly
Focused on availability, queueing, session tracking, wallet, menu, promos, and feedback
Admin Panel
Designed for GGX staff
Desktop-first control center
Focused on monitoring, approvals, PC assignment, wallet top-up, reports, promos, feedback, and incident tracing
Product Principles
1. Visibility First
Users should always know what is happening.
Examples:

Queue count is visible

Remaining time is visible
Wallet balance is visible
Admin actions update in real time

Players should not need to ask staff for every basic question.

View promos

3. Admin Has Final Control
The system gives freedom to users, but critical actions remain controlled by staff.
Examples:

PC assignment is admin-controlled
Wallet credits are admin-loaded
Session extensions can be approved or restricted
Maintenance mode is admin-managed
4. Real-Time System Behavior
The system should update without needing manual refresh.
Examples:
PC availability changes live
Queue movement updates live
Session timers update live
Admin dashboard updates live
Feedback and orders appear live
5. Error Prevention
The system should prevent mistakes before they happen.
Examples:
Prevent duplicate queue entries
Confirm session ending
Warn before time expires
Validate login and QR check-in
Restrict admin-only actions
Log important actions
